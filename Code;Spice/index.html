About the Program

Code;Spice is a coding program completely free of charge for kids of all ages that would like to learn Python, C++, Java, Web development, algorithms, or many other topics. We know that there are many other coding classes that are provided online, but we provide free, live, online classes that are based off of our own philosophy of learning programming that allowed for us to succeed. We want to teach our unique approach to students in order to allow them to succeed as well. We also offer office hours for struggling students in need of more help outside of class. We are very flexible schedule wise, so we will try to accomodate your schedule to the best of our ability, whether it is regarding class frequency or the dates and times of the classes. Recordings of classes will be posted for participants that missed classes on request, so don't worry about being a latecomer or missing any classes! In the end, we hope for participants to not only learn how to code, but to learn how to code more efficiently, and most importantly, have fun!

COURSES OFFERED FOR SPRING 2022 FIRST SEMESTER

Hover over any of the below courses to see more details about what is taught in each course. All courses are free unless otherwise specified. In addition, courses will be around 1-3 months long. Even if you will not be able to make the classes, stay tuned for courses for the second semester of summer that will begin later, or if courses haven't started too soon, you can catch up with the recordings we post. If you have any questions, feel free to email codespiceofficial@gmail.com.

Beginners Python

This class is meant to teach students the basics of Python. Python is a popular, useful, and easy language to learn that is very suitable for beginners. If you are not familiar with Python or have never programmed before, then this course is perfect for you. We will learn about syntax, different Python tools, and basic problem solving.

Advanced Python

In this class, we will apply Python in more involved settings. This class will be more project based. A strong background and understanding of Python is recommended. Initially, instructors will be leading and lecturing with project development, but soon, students will be completing their own projects with the guidance of instructors.

Basic C++

This class is meant to teach students the basics of C++. If you are not familiar with C++ or have never programmed before, then this course would be perfect for you. In this course, we will learn about syntax, different C++ tools, and basic problem solving.

Basic Java

This class will teach students the basics of Java. If you are not familiar with Java or have never programmed before, then this course would be perfect for you. In addition, this course is a strongly recommended prerequisite for AP Computer Science A. In this course, we will learn about syntax, different Java tools, and basic problem-solving techniques.

AP Computer Science Principles

This class will not only teach students the basics of Java, but also the fundamentals of programming itself. This class will be a lot more curriculum based (see here.) It is much more extensive and is designed to specifically prepare you for the AP CS Principles exam. This class is perfect for any complete beginners to programming.

AP Computer Science A

This class is an extension of the Basic Java class. It is much more advanced and is designed to prepare students for the AP Computer Science A exam/class (AP CS A) that they can take in high school (more information about the AP CS A can be found here). Proficiency with Java is strongly recommended for this class.

Private Classes

For private classes, both the time, frequency, class size, and topic of the classes are negotiated by instructor and students. A donation of 15 dollars/hour per person is REQUIRED. This donation will go into a Code;Spice bank account that is strictly meant to save money to donate to other worthy causes or support the further growth of the Code;Spice organization. We greatly appreciate any donations!

Basic Website Development

This class is meant to teach students the basics of developing websites with HTML and CSS. In this class, students will learn how to make their own websites, publish them, and various other decorative aspects of a website like navigation bars, carousels, galleries, etc.

Scratch Coding

This coding class is the most basic coding class available. It is most recommended for kids up to 5th grade that are completely new to coding. It teaches kids the basic coding philosophy and how coding works. It also introduces them to some basic concepts of coding.

Basic Algorithms

Basic Algorithms is a course meant to teach common basic algorithms such as Greedy algorithm, complete search, etc. This is meant to begin helping students develop an algorithmic mindset, pass USACO Bronze, or succeed in other similar competitions. A proficiency with Java, C++, or Python is strongly recommended.
