About the Program

Code;Spice is a coding camp completely free of charge for kids of all ages that would like to learn Python, relatively simple algorithmic thinking, how to get started on their own project, or just the coding philosophy in general. The classes are separated into two types of classes. The advanced class works on more complex projects like USACO, websites, games, you name it! The beginners class is more for reinforcing and getting proficient at Python. Throughout the school year, we will be meeting once a week for each type of class for 1.5 hours, in addition to optional 2 hour long office hours for both classes to help students with questions or problems. Recordings of classes will be posted for participants that missed classes, so don't worry about being a latecomer or missing any classes! In the end, we hope for participants to not only learn how to code, but to learn how to code more efficiently, and most importantly, have fun!

COURSES OFFERED FOR SUMMER 2022

Click on any of the below courses to see more details about what is taught in each course. All courses are free unless otherwise specified. If you have any questions, feel free to email codespiceofficial@gmail.com with any questions.

Beginners Python

This class is meant to teach students the basics of Python. Python is a popular, useful, and easy language to learn that is very suitable for beginners. It can be used to create many things, from games to artificial intelligence. If you are not familiar with Python or have never programmed before, then this course is perfect for you. We will learn about syntax, different Python tools, and basic problem solving.

Advanced Python

In this class, we will apply Python in more involved settings. This class will be more project based. A strong background and understanding of Python is recommended. Initially, instructors will be leading and lecturing with project development, but soon, students will be completing their own projects with the guidance of instructors.

Basic C++

This class is meant to teach students the basics of C++. C++ is a very popular language for competitive programming and can also be applied to make operating systems, games, and graphical user interfaces (GUIS). If you are not very familiar with C++ or have never programmed before, then this course would be perfect for you. In this course, we will learn about syntax, different C++ tools, and basic problem solving.

Basic Java

This class is meant to teach students the basics of Java. Java is also a very popular language for competitive programming and can also be applied for back-end development projects and Android development. If you are not very familiar with Java or have never programmed before, then this course would be perfect for you. In this course, we will learn about syntax, different Java tools, and basic problem-solving techniques.

AP Computer Science A

This class is an extension of the Basic Java class. It is much more advanced and is designed to prepare students for the AP Computer Science A exam/class (AP CS A) that they can take in high school (more information about the AP CS A can be found here). A good understanding and proficiency with Java is recommended for this class. Students will learn about more complex algorithms, techniques, etc.

Algorithms and Data Structures

The algorithms and data structures is a course meant to introduce and teach some of the easier and basic algorithms to students (for example, Greedy algorithm, complete search). This is meant to begin helping students develop an algorithmic mindset or pass USACO Bronze. A proficiency with a coding language, particularly Java, C++, or Python is strongly recommended for this class.

Private Classes

For private classes, both the time, frequency, class size, and topic of the classes are negotiated by instructor and students. A MANDATORY donation of 15 dollars/hour per person is REQUIRED. This donation will go into a Code;Spice bank account that is strictly meant to save money to donate to other worthy causes or support the further growth of the Code;Spice organization. We greatly appreciate any donations that support us!
